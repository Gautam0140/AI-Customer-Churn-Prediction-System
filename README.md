🧠 AI Customer Churn Prediction System

📌 Overview

Customer churn is a major problem for businesses. This project uses Machine Learning to predict whether a customer is likely to leave a company based on historical data.

▶️ How to Run

1. Clone the repository:
   git clone https://github.com/Gautam0140/AI-Customer-Churn-Prediction-System.git

2. Navigate to project folder:
   cd AI-Customer-Churn-Prediction-System

3. Install dependencies:
   pip install -r requirements.txt

4. Run the project:
   python main.py
